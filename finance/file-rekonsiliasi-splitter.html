How the ACMM Splitter works

1

Upload ACMM File

Upload the Transaction Summary Details Excel export from ACMM system.

2

Split by GL Code

Each unique GL Account Code (Col M) becomes its own sheet. Original sheet is preserved.

3

Net per Kode Rekon

Kode Rekon = Day + Store Code (from brackets).
Debit rows are summed; Credit rows are subtracted.

4

Download Output

Each GL sheet has Col A = Kode Rekon, Col B = Net Amount. Original data sheet also included.
